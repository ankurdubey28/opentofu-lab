# OpenTofu Lab

This repository is a small learning space for Infrastructure as Code with OpenTofu.

## Purpose

- Practice OpenTofu basics
- Experiment with IaC workflows
- Keep example configurations and notes in one place

## Prerequisites

- [OpenTofu](https://opentofu.org/) installed.
- An AWS account for the main examples, or another supported provider if you want to adapt the labs
- Local test setup, depending on the exercise

## Getting Started

1. Clone the repository.
2. Add or update your OpenTofu configuration files.
3. Run `tofu init`, `tofu plan`, and `tofu apply` as needed.

## Basics

- OpenTofu fundamentals
- Providers and resources
- Variables and outputs
- State basics
- Basic AWS examples
